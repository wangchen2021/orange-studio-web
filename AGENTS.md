## Style Guidelines (Sass Global Variables)
### Global Color Variables
- All global color variables are defined in the file: `config/style/sassVar.ts`
- Example of `sassVar.scss` content:
  ```typescript
  // config/style/sassVar.scss
export const sassVars = {
  primary: '#02B980',
  grassGreen: '#3DD27E',
  orange: '#FAAD14',
  error: '#FF3B30',
};
```
